# KPIs used for creating the dashboard

## Sales Performance (V-series)

| KPI ID   | KPI Name        | Description                                      |
|----------|----------------|--------------------------------------------------|
| KPI_V_1  | Total Sales    | Total number of cars sold                        |
| KPI_V_2  | Sales by Year  | Analysis of sales trends over time               |
| KPI_V_3  | Sales by Model | Comparison across different car models           |

## Revenue Analysis (R-series)

| KPI ID   | KPI Name              | Description                                      |
|----------|----------------------|--------------------------------------------------|
| KPI_R_1  | Total Revenue        | Revenue = Base Price × Sales Volume              |
| KPI_R_2  | Revenue by Model     | Revenue generated by each car model              |
| KPI_R_3  | Revenue by Region    | Revenue distribution across regions              |
| KPI_R_4  | Average Car Price    | Average selling price of cars                    |

## Product Analysis (P-series)

| KPI ID   | KPI Name                | Description                                      |
|----------|------------------------|--------------------------------------------------|
| KPI_P_1  | Sales by Fuel Type     | Distribution of sales by fuel type               |
| KPI_P_2  | Sales by Color         | Distribution of sales by car color               |
| KPI_P_3  | Sales by Horsepower    | Analysis of sales by horsepower range            |

## Geographic Analysis (G-series)

| KPI ID   | KPI Name                      | Description                                      |
|----------|------------------------------|--------------------------------------------------|
| KPI_G_1  | Sales by Region              | Total sales in each region                       |
| KPI_G_2  | Revenue by Region            | Total revenue generated in each region           |
| KPI_G_3  | Best-Selling Model by Region | Most sold car model in each region               |
